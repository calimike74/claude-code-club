# Module 01 — Budget & Cash Flow

**Level:** 1 (Baseline)
**Time:** 30-45 minutes + 30 minute challenge
**You'll need:** Claude Code installed and running

---

## What You'll Do

Build a 12-month cash flow forecast for a UK SaaS startup that's running out of money. Three scenarios, a break-even analysis, and a board-ready executive summary.

This is the first task a junior FP&A analyst does at any company. Before AI, it took a day. You'll do it in 30 minutes.

## Files in This Module

| File | What It Is |
|------|-----------|
| [brief.html](brief.html) | **Start here.** Open in your browser. Your mission briefing. |
| [helix-labs-financials.csv](helix-labs-financials.csv) | 3 months of real company data to work with |
| [prompts.md](prompts.md) | Step-by-step prompts for Claude Code |
| [challenge.md](challenge.md) | Independent mission — test yourself |

## How to Use This

1. Open `brief.html` in your browser — read the scenario
2. Open your terminal in this folder and start Claude Code
3. Follow the prompts in `prompts.md` step by step
4. When you're done, try the challenge in `challenge.md` without the guide

## What You'll Learn

- How to brief AI on a financial analysis task
- How to work with real data in Claude Code
- How to build scenario models (what-if analysis)
- How to challenge AI output and verify calculations
- How to write a board-level executive summary
