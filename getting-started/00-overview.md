# Claude Code Club — Resource Hub

## What This Is

A complete guide for running a Claude Code Club with sixth form students. Covers what to build, what tools to use, what free tiers to grab, and how to make sure they leave with something real.

---

## The Files

### [01 — What to Build](./01-what-to-build.md)
Project ideas at four levels, from single-session builds to ambitious portfolio pieces. Every project is achievable by non-coders using Claude Code, deployable on Vercel, and solves a real problem someone actually has.

### [02 — The Toolkit](./02-toolkit-free-services.md)
Every free service and API your students can use, organised by what they unlock. Covers databases, AI, email, SMS, maps, image handling, browser APIs, automation, and physical computing. All free or near-free.

### [03 — Student Free Tiers](./03-student-free-tiers.md)
Time-sensitive student deals and the critical "grab now vs grab at university" split. Includes GitHub Student Developer Pack, JetBrains, Figma, Notion, and cloud credits. Some of these expire or become harder to access after they leave school.

### [04 — Shipping and Beyond](./04-shipping-and-beyond.md)
The non-technical guidance that makes everything else matter. Shipping publicly, documenting work, building a GitHub profile, demo day, peer teaching, open source, and writing reference paragraphs. The builder mindset over tool literacy.

## The Core Stack

Every student in the club should be comfortable with this baseline:

| Layer | Tool | Cost |
|-------|------|------|
| Build | Claude Code | Included with Claude subscription |
| Store data | Supabase | Free tier |
| Add intelligence | Google AI Studio (Gemini) | Free |
| Deploy | Vercel | Free hobby tier |
| Version control | GitHub | Free |

Everything else in these documents extends this core stack.

---

## The One Rule

Every student leaves with **at least one deployed project** with a real URL they can share. Not on localhost. Not in a repo nobody will look at. A link they can text to someone.

---

## Context

- Students are upper sixth (Year 13), leaving in approximately three months
- Mix of backgrounds — not all are coders or plan to be
- Some heading to university, some heading into work
- Club uses Claude Code as the primary building tool
- All projects follow a build → deploy → share workflow
