# 03 — Student Free Tiers and Timing

## Why Timing Matters

Your U6 students are in a specific window. They're still students at school, which gives them some access. In three months, most will get a university .ac.uk email, which unlocks significantly more. A few are heading straight into work, where student deals disappear entirely.

This document splits into three sections:
1. **Grab this week** — while they're still at school
2. **Grab on matriculation day** — the moment they get a university email
3. **Free for everyone** — no student status required, no timing pressure

---

## Grab This Week (While Still at School)

### GitHub Student Developer Pack
**Value:** Thousands of pounds worth of tools, completely free.
**What's in it:** GitHub Pro, GitHub Copilot, cloud credits (DigitalOcean, Azure), MongoDB Atlas credits, free .me domain from Namecheap, JetBrains licence, and 100+ other tools.

**Eligibility:** Students aged 13+ enrolled in a course of study. Does not require a .edu or .ac.uk email — a school-issued email, student ID card, or enrolment letter should work. A .edu email speeds up verification, but alternative documentation is accepted.

**How to apply:**
1. Create a GitHub account (if they don't have one)
2. Go to [education.github.com/pack](https://education.github.com/pack)
3. Click "Get your pack"
4. Verify student status with school email or upload student ID

**Deadline pressure:** Current certification vouchers expire 30 June 2026. The pack itself doesn't expire while student status is verified, but there's no reason to wait.

**Action: Apply this week. Don't delay.**

---

### JetBrains Student Pack
**Value:** ~£600/year worth of professional IDEs, free.
**What's in it:** IntelliJ IDEA Ultimate, WebStorm, PyCharm Professional, and every other JetBrains IDE. All professional-grade tools used by working developers.

**Eligibility:** Students at accredited educational institutions, including secondary schools. Can verify via school email, ISIC card, or GitHub Student Developer Pack.

**How to apply:**
1. Go to [jetbrains.com/shop/eform/students](https://www.jetbrains.com/shop/eform/students)
2. Apply with school email, or authorise via GitHub if they already have the Student Developer Pack

**Note:** Licences are for non-commercial, educational use only. Renews annually while still a student. After graduation, JetBrains offers a 40% discount for two years.

**Relevance for your club:** Not essential for Claude Code work, but if any students are heading into computer science, having WebStorm or IntelliJ set up before freshers' week is a genuine advantage.

---

### Figma Education Plan
**Value:** ~£11/month Professional plan features, free for two years.
**What's in it:** Unlimited files, unlimited version history, collaboration with unlimited team members, prototyping, Dev Mode. Essentially the full Professional plan.

**Eligibility:** Students at any educational institution. Verify with student email.

**How to apply:**
1. Create a Figma account at [figma.com](https://www.figma.com)
2. Verify student status at [figma.com/education/apply](https://www.figma.com/education/apply)

**Note:** Verification lasts two years (recently changed — some users report one year, re-verify if needed). Even if they never become designers, Figma is the standard tool for mocking up app interfaces. Sketching a UI in Figma before building in Claude Code is a genuine workflow improvement.

**Caveat:** AI features are not included in the Education plan.

---

### Notion Education
**Value:** Plus plan (normally paid), free for students.
**What's in it:** Unlimited blocks, file uploads, 30-day page history, guest collaborators.

**Eligibility:** Students with a valid school or university email.

**How to apply:**
1. Sign up at [notion.so](https://www.notion.so)
2. Verify student status with school email

**Relevance for your club:** Useful as a project planning tool, documentation hub, or personal knowledge base alongside Claude Code and Obsidian.

## Grab on Matriculation Day (September — University Email)

The moment they receive their .ac.uk email, before they've unpacked, they should sign up for everything below. University emails unlock a completely different tier of freebies.

### Microsoft Azure for Students
- $100 in cloud credits, no credit card required
- Renews annually while they're students
- Useful for hosting anything more complex than Vercel allows

### Google Cloud for Students
- $300 in free credits for 90 days (standard new-user offer)
- Additional student-specific programmes available

### MongoDB Atlas (via GitHub Student Pack)
- Free tier is available to everyone (512MB)
- GitHub Student Pack gives additional credits
- Good for projects needing a document database rather than Supabase's relational approach

### Namecheap (via GitHub Student Pack)
- Free .me domain for one year
- Free SSL certificate
- firstname.me pointing at a Vercel project looks far more professional than a .vercel.app URL

### DigitalOcean (via GitHub Student Pack)
- Cloud credits for hosting
- More control than Vercel if they want to run a proper server

### Perplexity Pro Education
- Free for 12 months for verified students and educators
- Full Pro features including Study Mode, expanded citations, unlimited uploads
- Verify with academic email

### GitHub Copilot (upgrades via GitHub Student Pack)
- Verified students receive GitHub Copilot Student plan (enhanced over the free tier)
- Inline code suggestions in VS Code, JetBrains, and other editors
- Complements Claude Code — different tool, different strengths

## For Students Heading Into Work (Not University)

They lose the student-specific perks but keep the entire core toolkit. Vercel, Supabase, Resend, Cloudinary, Mapbox, Google AI Studio — all free for everyone.

Key advice for this group:
- **A deployed project in a job interview is worth more than any student discount.** Make sure they leave with a live URL.
- **The GitHub Student Developer Pack may still work** for a period after leaving school if they applied while still enrolled. But they should assume it won't and focus on the universally-free tools.
- **Vercel Pro is $20/month** if they need it later. That's the only meaningful cost they might hit, and the free tier is sufficient for most projects.
