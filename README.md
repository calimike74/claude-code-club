# Claude Code Club

Resources for running a Claude Code Club with sixth form students. Built by a practising teacher, tested with real students, and released for anyone to use.

---

## What This Is

A complete resource pack for running a Claude Code Club — a school-based programme where students learn to build real things using AI-assisted development. Covers two tracks:

- **Builders** — students who want to make websites, apps, and tools
- **Business & Economics** — students who want to apply AI to financial analysis, business strategy, and professional workflows

Everything here has been used in a real club with real students. The prompts have been tested, the project ideas have been built, and the free tiers have been verified.

## The Core Stack

Every student should be comfortable with this baseline:

| Layer | Tool | Cost |
|-------|------|------|
| Build | Claude Code | Included with Claude subscription |
| Database | Supabase | Free tier |
| AI | Google AI Studio (Gemini) | Free |
| Deploy | Vercel | Free hobby tier |
| Version Control | GitHub | Free |

## The One Rule

Every student leaves with **at least one deployed project** with a real URL they can share. Not on localhost. Not in a repo nobody will look at. A link they can text to someone.

## Using This Repo

### For students in the club
```bash
git clone https://github.com/YOUR_USERNAME/claude-code-club.git
```
Then open any file to get started. Pull before each session for updates:
```bash
git pull
```

### For teachers at other schools
Fork this repo and adapt it. The materials are designed for UK sixth form (Years 12-13) but the principles work at any level. Change the scenarios, swap the project ideas, adjust for your students.
